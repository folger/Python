String Functions
Char(number)$ Takes an integer 1-255, returns the ASCII character.example: char(65)$ returns A.
Code(str$) Takes a string, returns ASCII code for the first character.example: str$ = "abc"; code(str$) returns 97.
Compare(str1$, str2$ [,Case]) Takes two strings, returns 1 (identical) or 0 (not identical). Option Case controls case sensitivity: 1 = true (default), 0 = false. example: str1$ = "ABC"; str2$ = "abc"; compare(str1$,str2$,0) returns 1.
Exact(str1$, str2$) Takes two strings, returns 1 (identical, including case) or 0 (not identical). example: str1$ = "ABC"; str2$ = "abc"; exact(str1$,str2$) returns 0.
Find(within$, find$ [,StartPos]) Searches within$ for find$, returns the starting position. Option StartPos controls start search position (default = 1). Case sensitive. No wildcards. example: str1$ = "abcde"; str2$ = "bc"; find(str1$,str2$) returns 2.
Format(d[,sigFig$])$ Converts a double to string. Option sigFig$ controls significant figures; if not specified or empty string "" is used, uses @SD digits. Use "*" to use Origin's global setting. example: Format(2.01232,"*3")$ returns 2.01.
Left(str$, n)$ Takes a string, returns the leftmost n characters. example: str$ = "abcde"; Left(str$,3)$ returns abc.
Len(str$) Takes a string str$, returns the number of characters. example: str$ = "abc ABC"; Len(str$) returns 7.
Lower(str$)$ Takes a string str$, converts it to lowercase. example: str1$ = "ABCDE"; str2$ = Lower(str1$)$ returns abcde.
MakeCSV(str$[, quote, output_delim, input_delim$])$ Takes a delimited string, converts it to CSV. Option quote to enclose output: 0 (default) = no quotes, 1 = single quotes, 2 = double quotes. Option output_delim: 0 = comma, 1 = semicolon. Option input_delim$ specifies specifies source string delimiter (not needed if white space). example: str1$ = "This is a test value"; MakeCSV(str1$, 1, 0)$ returns 'This','is','a','test','value'
MatchBegin(within$,find$[,StartPos,Case]) Search string within$, return an integer corresponding to the start position of find$ or -1 (not found). Supports "*"' and "?" wildcards. Option StartPos specifies the position of the character at which to start the search: 1 (default) = search from 1st character. Option Case controls case sensitivity: 0 (default) = false, 1 = true. example: str1$ = "From: test@Originlab.com"; str2$ = "From*@"; MatchBegin(str1$,str2$,1); returns 1.
MatchEnd(within$, find$[, StartPos, Case]) Search string within$, return an integer corresponding to the end position of find$ or -1 (not found). Supports "*" and "?" wildcards. Option StartPos specifies the position of the character at which to start the search: 1 (default) = search from 1st character). Option Case controls case sensitivity: 0 (default) = false, 1 = true. example: str1$ = "From: test@Originlab.com"; str2$ = "From*@"; MatchBegin(str1$,str2$,1); returns 11.
Mid(str$, StartPos, n)$ Takes a string str$, returns n characters from StartPos. example: str$ = "aabcdef"; Mid(str$,2,3)$ returns abc.
Replace(within$, StartPos, n, replace$)$ Replace n characters in within$, starting from StartPos, with string replace$. String replace$ may differ in length from n. example: Replace(abcdefghijklmn,3,5,123456)$ returns ab123456hijklmn.
Right(str$, n) Takes a string str$, returns the rightmost n characters. example: str$ = "abcde"; Right(str$,n)$ returns cde
Search(within$, find$[, StartPos]) Finds string find$ within string within$. Not case sensitive. No wildcards. Option StartPos controls where to start search (default = 1). example: within$ = "abcde"; find$ = "BC"; Search(within$,find$) returns 2.
Substitute(within$,sub$,find$ [, n])$ Search string within$ for string find$, replace with sub$. Option to substitute only the nth found instance. example: Substitute(abcdefabcdef,12,bcd,0)$ returns a12efa12ef
Token(str$,iToken[, iDelimiter])$ Takes string str$, returns substring corresponding to index iToken. Option iDelimiter is ASCII value of delimiter: 0 (default) = any white space; 32 = single space; 124 = |). example: str1$="This is my string"; Token(str1$,3)$ returns my.
Trim(str$[, n])$ Takes string str$ and removes spaces. Parameter n controls how space is removed: 0 (default) = leading + trailing, 1 = remove all. example: str1$ = " abc ABC "; Trim(str1$,0)$ returns abc ABC.
Upper(str$)$ Takes string str$, returns as uppercase. example: str1$ = "abcde"; Upper(str1$)$ returns ABCDE
Value(str$) Takes string number str$, returns it as double. example: str$ = "+.50"; Value(str$) returns 0.5. Note: see the atof() function.

Date and Time Functions
Date(MM/dd/yy[, format$]) Takes a date-time string and returns Julian-date value. example: date(24-09-2009,"dd-MM-yyyy") returns 2455098. If format$ is not specified, string is interpreted using system short date format. example: date(3/5/14) returns 2456721 (US) but date(5/3/14) returns 2456721 (UK).
Date2str(d,format$)$ Takes a Julian-date value and returns a date string. example: date2str(2456573.123, "dd/MM/yyyy HH:mm")$ returns 08/10/2013 02:57).
Day(d, [,n]) Takes a Julian-date value, returns the day number. If option n = 1, returns 1 to 31 (month); if n = 2, returns 1 to 366 (year). example: (Day(2454827.5982639, 2) returns 362.
Hour(d) Takes a Julian-date value, returns the hour as an integer. Returns 0 to 23 (0 = 12:00 A.M., 23 = 11:00 P.M.). example: Hour(0.6997854) returns 16.
Minute(d) Takes a Julian-date value, returns the minutes as an integer (0 to 59). example: Minute(2454827.5982639) returns 21.
Month(d) Takes a Julian-date value, returns the month as an integer (0 to 12). example: month(2454821) returns 12.
MonthName(d[,n])$ Takes a Julian-date value, returns the month name. Month format specified by option n: 1(single character), 3(3 characters = default), 0 (full month name), or -1 (returns 3 character English regardless of language settings). example: MonthName(2454827.5982639, 0)$ returns December.
Now() Returns the current date-time as a Julian-date value.
Quarter(d) Takes a Julian-date value, returns current quarter of the year.example: Quarter(2454829.5745718) returns 4.
Second(d,[b]) Takes a Julian-date value or real number, returns the seconds as a real value in the range 0 to 59.9999... example: second(2454827.5982639) returns 30.00.
Time(HH,mm,ss)/Time(HH:mm:ss[,Format$]) Takes HH,mm,ss or HH:mm:ss or date-time string and returns the Julian-date value.
Time2str(d,format$)$ Takes a Julian-date value, returns a time string of a specified format. example: time2str(0.1875, "HH:mm")$ returns 04:30.
Today() Returns the current date as a Julian-day value.
WeekDay(d[,n]) Takes a Julian-date value, returns the day of the week. Option n specifies week start and end values: 0 (default) = Sunday (0-6), 1 = Sunday (1-7), 2 = Monday (0-6), or 3 = Monday (1-7) example: weekday(2454825, 1) returns 5.
WeekDayName(d[,n1,n2])$ Takes a Julian-date value, returns the weekday. Options to specify output string length and week start and week end values. example: WeekDayName(2454825,-1,0)$ returns THU.
WeekNum(d[,n]) Takes a Julian-date value, returns the calendar week number of the year (1 to 53). Option to specify week start (Sunday vs Monday). example: weeknum(date(1/11/2009)) returns 3.
Year(d) Takes a Julian-date value, returns the year as an integer(0100-9999). example: year(2454821) returns 2008.
YearName(d[,n])$ Takes a Julian-date value, returns year as a string. Form of string is specified by option n: 0 = 2 digits, 1(default) = 2 digits preceded by an apostrophe, or 2 = 4 digits. example: YearName(2454827.5982639, 1)$ returns '08.
Statistical Functions
Count(vd[,n]) Takes a vector vd, returns the number of elements. Option n specifies element: 0 (default) = all; 1 = numeric; 2 = missing. example: count(col(a),2) might return 22 for number of missing values.
cov(vx, vy, avex, avey) Takes datasets vx and vy and respective means avex and avey, returns the covariance. example: cov(book1_b,book1_c,0.40302,0.38604) might return 0.0024571550460084.
Max(vd) Takes a range, dataset or column vd, returns the maximum value. example: max(col(A)) returns max value in col(A); max(1,2,3,4,9) returns 9.
Mean(vd) Takes a range, dataset or column vd, returns the average. example: mean(col(A)) returns the average value of col(A).
Median(vd[,method]) Takes a worksheet column vd, returns the median. Option method specifies the interpolation method: 0 (default) = empirical distribution with averaging; 1 = nearest neighbor; 2 = empirical distribution; 3 = weighted average right; 4 = weighted average left; 5 = Tukey hinge). example: median(col(A),2) returns the median (as determined using n = 2). For more on interpolation methods, see Interpolation of Quantiles.
Min(vd) Takes a vector vd, returns the minimum value.
rms(vd) Takes a vector vd, returns the root mean square.
Ss(vd [,ref]) Takes a vector vd, returns the sum of squares. Sum of squares is calculated after subtracting some reference value ref from each value in vd. Option ref defaults to the mean of vd but ref can be a constant, a dataset or a function. example: ss(vd) returns the mean-subtracted sum of squares.
StdDev(vd) Takes a vector vd, returns the sample standard deviation. example: StdDev(col(A)) returns sample standard deviation.
StdDevP(vd) Takes a vector vd, returns the population standard deviation. example: StdDevP(col(A)) returns population standard deviation.
Total(vd) Takes a vector vd, returns the sum of elements. example: total(col(a)) returns the sum of all data points in column A.
sumif(vd, vcon) Takes a vector vd, returns sum of values satisfying condition vcon.
countif(vd, vcon) Takes a vector vd, returns the count of values satisfying condition vcon.
averageif(vd, vcon) Takes a vector vd, returns the mean of values satisfying condition vcon.
ave(vd, size) Takes a vector vd, returns a range of averages of each group of size. If elements of vd is not an even multiple of n, then the returned average will represent only mod(vdSize,size) elements. example: ave(col(a),5) breaks col(a) into groups of size 5 and calculates an average for each group.
diff(vd) Takes a vector vd, returns a range of differences between adjacent elements. The first element in the returned range is vd(i+1)-vdi, and so forth. If there are N elements in vd, the returned range will contain N-1 elements.
sum(vd) Takes a vector vd, returns a range whose i+1th element is the sum of the first i elements. The last element of the returned range is the sum of all elements in the dataset.
Confidence(alpha, std, size) Takes significance level alpha, population standard deviation, and sample size, returns the confidence interval for the population mean. example: confidence(0.05, 1.5, 100) returns 0.29399459768101.
histogram(vd, inc, min, max) Takes a dataset vd, bin width = inc, vd min and vd max, and generates data bins. Data points that fall on the upper edge of a bin are placed into the next higher bin.
Kurt(vd) Takes a vector vd, returns the kurtosis. example: dataset ds = {1, 2, 3, 2, 3, 4, 5, 6, 4, 8}; kurt(ds) returns 0.39502164502164.
Percentile(vx, vy) Takes a dataset vx, returns the percentile values at each percent value specified in vy. example: DATA1_A = normal(1000); DATA1_B = {1, 5, 25, 50, 75, 95, 99}; DATA1_C = percentile(DATA1_A, DATA1_B); returns a dataset DATA1_C that contains the percentiles of a normal distribution at 1%, 5%...99%.
QCD2(n) Takes a sample size n, returns the Quality Control D2 Factor. example: QCD2(4) returns 2.05875.
QCD3(n) Takes a sample size n, returns the Quality Control D3 Factor. Factor D3 is the 3-sigma lower control limit in the X bar R chart. example: QCD3(10) returns 0.223.
QCD4(n) Takes a sample size n, returns the Quality Control D4 Factor. Factor D4 is the 3-sigma upper control limit in the X bar R chart. example: QCD4(10) returns 1.777.
Skew(vd) Takes a dataset vd, returns the skewness (distribution asymmetry). examples: skew(col(a)) returns the skewness of the dataset in column A.
Emovavg(vd,n[,method]) Takes a vector vd, an integer n = time period, returns a vector of exponential moving averages. Option method specifies where calculation begins: 0 (default) starts from point n, and example: for(ii=1;ii=30;ii++) col(1)[ii] = ii; col(3)=emovavg(col(1),10, 1); //method II fills column 3 with values, calculated using starting point = 1.
Mmovavg(vd,n) Takes a vector vd, an integer n = time period, and returns a vector of modified moving averages. example: for(ii=1;ii=30;ii++) col(1)[ii] = ii; col(2)=mmovavg(col(1),10); fills column 2 with a modified moving average value at each point, starting with row 10.
Movavg(vd,back,forward) Takes a vector vd, a backward and forward offset, and returns vector of adjacent averages. Backward and forward offset are with respect to the current row number. example: for(ii=1;ii=10;ii++) col(1)[ii] = ii; col(2)=movavg(col(1),0, 2); fills col(2) with adjacent average values at each point (Note that col(2)[9] = (col(1)[9]+col(1)[10])/2 and col(1)[10] = col(2)[10]).
Movslope(vx,vy[,n]) Takes two vectors, vx (independent) and vy (dependent), returns a vector of moving slope at each point. Optional n specifies window width (should be 1). If n is even, 1 will be added. When n is not specified, the function returns a vector of one value which is the linear fit slope of the input. example: col(C)=movslope(col(A),col(B),5); fills col(C) with slope at each point (first and last two cells are missing values).
Tmovavg(vd,n) Takes a vector vd, an integer n = time period, returns a vector of triangular moving averages. example: for(ii=1;ii=30;ii++) col(1)[ii] = ii; col(2)=tmovavg(col(1),9); fills col(2) with triangular moving average values at each point, starting at point = 9.
Wmovavg(vd,vw) Takes a vector vd (data to smooth), vector vw (indexed weight factor), returns a vector of weighted moving averages. example:
for(ii=1;ii=30;ii++) col(1)[ii] = ii; //data vector
for(ii=1;ii=10;ii++) col(2)[ii] = ii/10; //weight vector
col(3)=wmovavg(col(1),col(2));
fills col(3) with the weighted moving average at each point, starting at point = 10.

Data Generation Functions
Data(x1,x2,inc) Takes two values x1 and x2 and creates a dataset, ranging from x1 to x2, with an increment, inc. If x1 = x2, function returns inc number of points with values = x1. Default for inc = 1. example: col(A) = data(0,100,5) fills column A with numbers from 0 to 100, by increment = 5; .
grnd() Returns a value from a normally (Gaussian) distributed sample, with zero mean and unit standard deviation. The initial value and sequence of values are the same for each Origin session. No argument is needed. Commonly, the function is used to return a random value from a normal distribution of known mean and standard deviation, using the following expression: grnd()*sd+m. example: aa=grnd()*0.30855+0.45701 might return 0.33882089669989.
normal(npts[,seed]) Returns a range of npts. Values are random numbers with normal distribution (zero mean, unit standard deviation). If seed is omitted, a different seed is used each time the function is called. example: col(1) = normal(100)*2+5 fills column 1 with 100 random values with mean = 5 and sd = 2.
Poisson(n, mean [,seed]) Returns n random integers having a Poisson distribution with mean. Optional seed provides a seed for the number generator. example: col(1)=Poisson(100,5,1) fills column 1 with 100 random values having a Poisson distribution with mean of 5.
